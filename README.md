# GuoFanBlog

GuoFanBlog 是一个基于 Hexo 的个人博客项目。

## 项目结构
- `source/`：博客内容和文章目录
- `themes/`：主题文件夹（如 butterfly）
- `scaffolds/`：文章、页面模板
- `_config.yml`：Hexo 配置文件
- `db.json`：Hexo 数据库文件
- `LICENSE`：开源协议
- `package.json`：依赖和脚本

## 快速开始
1. 安装依赖：
   ```bash
   npm install
   ```
2. 启动本地服务：
   ```bash
   npx hexo server
   ```
3. 生成静态文件：
   ```bash
   npx hexo generate
   ```

## 主题
本项目使用 [butterfly](https://github.com/jerryc127/hexo-theme-butterfly) 主题。
